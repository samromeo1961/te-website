Technical Specifications

Built with modern technologies for reliability and performance

System Requirements

  • OS: Windows 10 or later (64-bit)
  • RAM: 4GB minimum, 8GB recommended
  • Storage: 500MB for application
  • Display: 1280x720 minimum

Database Requirements

  • SQL Server: 2016 or later
  • Databuild: System + Job databases
  • Connection: TCP/IP enabled
  • Auth: SQL Server authentication

Technology Stack

  • Framework: Electron + Vue.js 3
  • Data Grid: AG Grid Community
  • PDF Engine: Puppeteer
  • Updates: Auto-update via GitHub

Security

  • Credentials: Encrypted local storage
  • Queries: Parameterized (SQL injection safe)
  • Updates: Signed releases
  • Database: Read/write, no schema changes

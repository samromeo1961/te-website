Enterprise-Grade Estimating

Built for professional quantity surveyors and estimating teams

BIM Integration

Import IFC files directly and extract quantities from BIM models. Link cost data to model elements for visual cost planning.

PDF Takeoff

Full-featured 2D takeoff from PDF drawings. Measure areas, lengths, and counts with automatic calculations and assemblies.

Flexible BOQ

Create bills of quantities in any format - trade-based, elemental, or location-based. Full support for variations and revisions.

Resource Libraries

Build comprehensive libraries of materials, labour, plant, and subcontract items. Share across projects and teams.

Cost Analysis

Analyse costs by element, trade, or any custom breakdown. Compare estimates, track revisions, and identify cost drivers.

Team Collaboration

Multi-user environment with role-based permissions. Multiple estimators can work on the same project simultaneously.
